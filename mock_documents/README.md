mock_documents/

This folder contains mock `.txt` documents intended for ingestion testing with the DecisionsRX RAG MVP.

Files:
- scientific_study.txt — scientific-style abstract + findings
- business_analysis.txt — executive/business summary
- supply_chain_note.txt — vendor/lead-time notes
- timeline.txt — deployment timeline
- regulatory_summary.txt — regulatory overview
- mixed_evidence.txt — mixed/conflicting evidence used to test handling of contradictions
- institutional_note.txt — example institutional note (indexed like documents)

To convert any `.txt` to `.pdf` or `.docx` for additional format testing, use your preferred tools (LibreOffice, pandoc, Word).
